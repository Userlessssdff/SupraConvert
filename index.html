About UltraConvert

Open source. Lightweight. Secure. Built for modern workflows.

What is this tool?

UltraConvert is a specialized, lightweight web interface for the powerful Python Pillow library. It allows you to perform complex image manipulations‚Äîformat conversion, high-quality resizing, and algorithm-based compression‚Äîwithout needing to use the command line or install heavy desktop software like Photoshop. It focuses on providing a fast, secure, and privacy-focused conversion utility.

Key Features & Capabilities

  • Batch Conversion Ready: Built with a job-queue architecture to handle multiple large files concurrently.
  • High-Fidelity Resizing: Uses the advanced LANCZOS resampling filter for superior image quality during scaling.
  • Lossy & Lossless Compression: Fine-tune JPG/WebP quality from 1% to 100% for optimal size/quality balance.
  • Transparency Support: Full support for PNG, GIF, and WebP transparency, including automatic compositing over a white background when converting to opaque formats (JPEG/BMP).
  • Opacity Adjustment: Unique feature to control the overall transparency of an image before saving.
  • Secure & Self-Cleaning: All uploaded files and conversion outputs are automatically deleted from the server after a short period (30 minutes) for user privacy.

Architecture and Technology Stack

üêç

Powered by Python/Flask

The backend runs on a lightweight Flask server backed by Python 3. It manages file uploads, job queuing, and API routing. This minimal setup ensures low resource usage and high stability.

‚öôÔ∏è

Thread-Safe Processing

Image processing is delegated to a multi-threaded worker pool (ThreadPoolExecutor), ensuring the main web server thread remains responsive, even during long conversions. A global lock (`JOB_LOCK`) maintains the integrity of the job queue state.

‚ö°

Asynchronous Frontend

The frontend is built with pure HTML5 and vanilla JavaScript/CSS. It relies entirely on the Fetch API for all communication, enabling a modern, single-page application (SPA) feel without the overhead of heavy frameworks.

üîí

Security & Performance

The application includes built-in security features such as rate limiting (via Flask-Limiter), file type validation (using `python-magic`), and secure, token-based filenames to prevent directory traversal and unauthorized access to output files.
